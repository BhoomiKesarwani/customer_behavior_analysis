# 🛍️ Customer Shopping Behavior Analysis

An end-to-end **Data Analytics Project** that analyzes customer shopping behavior using **Python, MySQL, and Power BI**. The project includes data cleaning, exploratory data analysis (EDA), SQL queries, and an interactive Power BI dashboard to uncover customer purchasing patterns and business insights.

## 🚀 Tech Stack
- Python (Pandas, NumPy, Matplotlib)
- MySQL
- Power BI
- Git & GitHub

## 📊 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- SQL-Based Business Analysis
- Interactive Power BI Dashboard
- Customer Segmentation
- Sales & Purchase Trend Analysis

## 📂 Project Workflow
1. Load dataset using Python
2. Clean and preprocess data
3. Perform EDA
4. Run SQL queries for business insights
5. Build an interactive Power BI dashboard

## 📈 Key Insights
- Customer demographics and buying patterns
- Product category performance
- Revenue and sales trends
- Payment method analysis
- Subscription and repeat customer analysis
